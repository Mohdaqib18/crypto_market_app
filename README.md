# crypto_market_app
Displaying trends of crypto assets. (live link: https://aqib-frontend-dev-crypto.netlify.app/ )

# Instructions
1. git clone https://github.com/Mohdaqib18/crypto_market_app.git
2. npm install (in root directory)
3. npm run dev
